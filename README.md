# SimpleAuction

## Depends on

* truffle
* ganache-ui
* ethers
* react
* @mui/material


## Getting started

```
$ git clone https://github.com/ChanMo/simple-auction
$ cd simple_auction
$ npx ganache-cli # or use ganache-ui
$ npx truffle build
$ npx truffle migrate
$ cd client
$ npm install
$ npm run start

```
